Inferential Thinking 01

The Foundations of Data Science

Inferential Thinking is an introductory text for data science that explores foundational concepts in data processing and statistics, using modern programming tools. Ideas are illustrated using real-world data sets. While rigorous in presentation, this text does not expect prior experience in computing, calculus, or linear algebra.
